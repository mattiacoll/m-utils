# m-utils
Some simple shorthands for some js functions and methods

## Install

// todo install (possibly npm)

```
npm i -D m-utils
```

## Usage

Just import the needed functions into your file

```
import { addEvent, byId } from 'm-utils'

addEvent( byId( 'some-selector' ), 'click', ( ev ) => {
  console.log( 'click' );
});
```
